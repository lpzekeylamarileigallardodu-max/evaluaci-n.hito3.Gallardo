# Evaluación Estructura de Datos
Proyecto para realizar el gestionamiento de notas, asientos y estudiuantes de una instituciòn educativa.
# Endpoints para verificar en funcionamiento
- /notas/agregar?valor=45 — agrega una nota
- /notas — ver todas
- /notas/promedio — promedio
- /notas/max y /notas/min — nota mayor y menor
- /aula — ver matriz de asientos
- /aula/ocupar?fila=3&columna=1 — ocupar asiento
- /aula/liberar?fila=2&columna=2 — liberar asiento
- /aula/ocupados — cuántos asientos ocupados
- /estudiantes/agregar?id=5&nombre=Mussolini&puntaje=100 — agregar estudiante
- /estudiantes — ver todos
- /estudiantes/ordenados — aprobados de mayor a menor puntaje
- /estudiantes/buscar/Ana — buscar por nombre
